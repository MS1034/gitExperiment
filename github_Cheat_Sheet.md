# Git Hub Cheat Sheet

## Basics

###### Check the Version

```bash
git --version
```

###### Create an empty Git repository:  

```bash
git init
```

###### Show modified files in working directory

```bash
git --version
```

###### Add a file as it looks now to your next commit (stage)

```bash
git add <file/directory name #1> <file/directory name #2> < ... >
```
